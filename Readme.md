# MicroIPTool

MicroIPTool is a simple graphical network utility for Windows, designed to simplify everyday network configuration and diagnostics.

## ⚙️ Features

* List available network adapters
* Display adapter information
* Configure IP settings:

  * DHCP mode
  * Static IP address
* Ping a specified IP address
* Basic network scanning (ping sweep)
* Multi-language interface:

  * Hungarian
  * English
* Error logging to file

## 📌 Status

Version: **0.8 (Beta)**

The application is already usable, but still under development. Minor bugs may occur.

## 💻 Requirements

* Windows 10/11 operating system
* Administrator privileges (required for modifying IP settings)

## ▶️ Usage

No installation required.

1. Run the `.exe` file
2. Select the desired network adapter
3. Configure IP settings or use ping / scan functions

## 🌐 Language

The application supports both Hungarian and English.
Language can be changed from the menu.
